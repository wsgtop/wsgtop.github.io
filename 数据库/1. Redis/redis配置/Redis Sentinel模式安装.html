Redis Sentinel 部署到服务器

哨兵模式，是最常见的，也是官方推荐的高可用性的解决方案。

单节点下载安装

  1. 下载地址

    wget https://download.redis.io/releases/redis-6.2.4.tar.gz
  2. 解压到当前目录下的redis文件内

    tar -xvf redis-6.2.4.tar.gz -C ./redis
  3. 安装make，二进制对应包

    yum -y install gcc automake autoconf libtool make

    yum install gcc gcc-c++
  4. 切换到解压目录并进行安装

    cd redis && makr
  5. 启动服务并测试

    启动服务 ： nohup ./src/redis-server &
    客户端连接： ./src/redis-cli
    命令测试：

6.没有报错，证明安装使用无误
