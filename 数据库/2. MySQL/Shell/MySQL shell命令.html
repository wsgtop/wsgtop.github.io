MySQL启停
启动 MySQL: net start MySQL80
关闭 MySQL: net stop MySQL80

连接MySQL
mysql -h 127.0.0.1 -u root -p

  • 创建新用户：

    username为自定义的用户名；

    host为登录域名，host为’%’时表示为 任意IP，为localhost时表示本机，或者填写指定的IP地址；

    paasword为密码

create user 'username'@'host' identified by 'password';

  • 用例：
    create user ‘test1‘@’%’ identified by ‘Test@1234’; – 创建用户

    drop user test1; – 删除用户

    flush privileges; – 授权之后刷新权限：

    Grant all privileges ON . TO ‘slave‘@’%’; – 授予所有权限

    show grants for slave; – 查看用户权限

    grant all privileges on ‘databasename’.’tablename’ to ‘username‘@’host’; – 为用户授权：
    • 其中*.第一个表示所有数据库，第二个表示所有数据表，如果不想授权全部那就把对应的写成相应数据库或者数据表；username为指定的用户；%为该用户登录的域名
