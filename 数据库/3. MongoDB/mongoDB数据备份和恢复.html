备份数据

基本语法

mongodump -h dbhost -d -dbname -o dbdirectory

  • -h MongoDB所在服务器地址，例如127.0.0.1 或者127.0.0.1:27017
  • -d 数据库需要备份的数据库实例 例如：Daguang
  • -o 数据库备份文件存放地址 例如 c:\Users\Administrator\Desktop\123

备份实例
在桌面新建一个名为123的文件夹，在数据库的bin目录下执行命令，

mongodump -h 127.0.0.1 -d Daguang -o c:\Users\Administrator\Desktop\123


执行完毕后数据文件会保存123文件的Daguang文件夹中

恢复数据

基本语法

mongorestore -h <:port> -d dbname

执行命令

mongorestore.exe -h 127.0.0.1:27017 -d ele C:\Users\Administrator\Desktop\ele
将本地的备份数据恢复到数据库ele中
