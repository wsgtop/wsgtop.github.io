1. MongoDB基本命令

1.1. Window

在安装的bin目录下使用命令 mongod.exe --config mongod.cfg

进入mongodb shell ，在bin目录下使用命令 mongo.exe

1.1.1. 基本命令

在mongo shell中使用命令时，后面的分号有没有都可以
连接到指定的数据库：mongo 127.0.0.1:27017/Daguang

help命令：help

显示数据库名称：show dbs

切换或者新建指定数据库：use <db_name>

显示当前所在数据库：db

显示当前库集合名称：show collections 效果等同 show tables

显示当前库用户名称：show users

显示日志：show logs

显示系统配置：show profile

退出mongo shell：exit

新增数据库： use <db_name> 示例: use test_db

新增集合：db.createCollection(name,options) 示例： db.createCollection(“user”)

删除数据库：db.dropDatabase() 删除当前数据库

删除集合：db.collection.drop()

1.1.2. 新增文档

添加数据：db.<collection_name>.insert(document) 示例: db.test_db.insert({“name”:”张三”})。若集合不存在，则新建后添加

更多用法：

insertOne()：插入一个

insertMany()：插入多个

1.1.3. 删除文档

删除文档： db.<collection_name>.remove(<query>,{justOne,writeConcern})

示例：db.test_db.remove({‘name’:’张三’}) 删除collection中名称为张三的文档数据

清空文档： db.<collection_name>.remove({})

1.1.4. 更新文档

更新文档： db.update(<query>,<update>,{upsert,multi,writeConcern})

示例：db.test_db.update({‘name’:’张三’},{$set:{‘age’:’19’}})

1.1.5. 查询文档

查看文档： db.<collection_name>.find()

示例：

db.test_db.find()

db.test_db.find({‘name’:’张三’})

db.test_db.find({‘age’:{$gt:’5’})

db.test_db.find({‘age’:{$gt:’5’}).pretty()
