1. Linux下 MongoDB安装

MongoDB 下载地址

1.1. Linux tgz安装

1.1.1. 选中对应的软件包

选择tgz，shell(tgz)是MongoDB 的shell客户端，并非服务端

1.1.2. 下载

拷贝下载链接，命令 wget https://fastdl.mongodb.org/linux/mongodb-linux-x86_64-rhel80-4.4.6.tgz下载到服务器

1.1.3. 解压安装

解压mongodb tar -xvf mongodb-linux-x86_64-rhel80-4.4.6.tgz

放在/opt/mongodb目录下，并在次目录下新建日志目录logs ,新建数据存储目录data；

新建配置文件 mongod.cfg(配置文件的文件名可以随意起)，放在bin目录下

1.1.4. 启动并配置环境变量

在bin目录下运行命令 mongod --config mongod.cfg启动MongoDB

运行mongo命令进入shell页面进行命令操作

vim /etc/pfofile 进入系统配置文件，在文件末行追加环境配置。不同的服务配置之间使用冒号分割

示例： export PATH=$PATH:/opt/node/bin:/usr/local/MongoDB/bin

source /etc/profile 使配置文件立即生效。即可在全局使用命令

1.2. linux RPM方式安装

系统

1.2.1. 下载MongoDB

1.2.2. 安装MongoDB

将上传的RPM包上传到linux服务器指定的位置，本次上传的位置是/opt/MongoDB/目录下

使用命令 rpm -ivh mongodb-org-server-4.4.6-1.el8.x86_64.rpm安装mongodb

1.2.3. 查看安装是否成功

systemctl status mongod 查看服务状态

systemctl start mongod 启动服务

systemctl stop mongod 停止服务
