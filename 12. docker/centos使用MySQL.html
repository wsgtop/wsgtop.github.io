CentOS 7 使用MySQL

查看系统版本

  • uname -a
  • uname -r
  • cat /etc/issue
  • cat /etc/redhat-release

下载并安装MySQL

  1. 在官网找到对应CentOS服务器的yum仓库rpm包下载

  1. 将下载的RPM安装包上传到服务器或者使用wget下载
    wget下载 wget -i -c https://dev.mysql.com/get/mysql80-community-release-el7-3.noarch.rpm
    安装 yum -y install mysql80-community-release-el7-3.noarch.rpm
    然后进行安装MySQL-Server ：yum isntall mysql-server

  2. 运行MySQL
    查看MySQL状态： systemctl status mysqld.service
    启动MySQL状态： systemctl start mysqld.service
    停止MySQL状态： systemctl stop mysqld.service
    重启MySQL状态： systemctl restrt mysqld.service

  3. 连接MySQL

  • 连接MySQL：首先找到MySQL密码，查看日志，找到安装时默认的随机密码，输入连接命令，登录成功。
  • 测试命令。输入命令，提示必须要重置密码。
  • 更改密码：ALTER USER 'root'@'localhost' IDENTIFIED BY 'MySQL@1234';
  • 测试命令。此时，输入命令执行成功。
