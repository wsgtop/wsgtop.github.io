Docker使用

MySQL

  • 通过命令拉取 MySql 镜像 docker pull mysql
  • 运行 MySql 容器 docker run -itd -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 --name mysql-test mysql
    • 命令说明：运行 MySQL，给定别名 mysql-test，容器内端口 3306 映射到主机端口 3306，设置 root 用户密码为 123456
  • 查询正在运行的容器 docker ps
  • 查询所有容器 docker ps -a
  • 查询本地镜像 docker images
  • 启动/关闭/重启容器 docker start/stop/restart [CONTAINER ID/NAMES]
  • docker 进入 MySQL 容器内部
    命令：docker exec -it mysql-test bash
    命令说明：在容器名为 mysql-test 中运行 bash 脚本

ubuntu

  • 拉起镜像 docker pull ubuntu 或 docker pull ubuntu:latest
  • 查看本地镜像 docker images
  • 运行ubuntu容器 docker run -itd --name ubuntu-test1 ubuntu
  • 进入容器内部 docker exec -it ubuntu-test1 bin/bash
  • 关闭容器 docker stop ubuntu-test1
  • 启动容器 docker start ubuntu-test1
  • 查看容器 docker ps 或者 docker ps -a

Nginx

  • 拉取镜像 docker pull nginx 或 docker pull nginx:latest
  • 运行容器（本地 8080 端口映射到容器内部的 80 端口） docker run --name nginx-test -p 8080:80 -d nginx

redis

  • 取最新版的 Redis 镜像 docker pull redis:latest
  • 运行容器 docker run -itd --name redis-test -p 6379:6379 redis
  • 进入容器内部 docker exec -it redis-test /bin/bash
