　在 Markdown 文档中，可以直接采用 HTML 标记插入空格（blank space），而且无需任何其他前缀或分隔符。具体如下所示：

插入一个空格 (non-breaking space)
&nbsp; 或 &#160; 或 &#xA0;

插入两个空格 (en space)
&ensp; 或 &#8194; 或 &#x2002;

插入四个空格 (em space)
&emsp; 或 &#8195; 或 &#x2003;

插入细空格 (thin space)
&thinsp; 或 &#8201; 或 &#x2009;

　　注意：不要漏掉分号。
