node.js安装

centos7 二进制安装

  1. 下载二进制文件上传到指定的位置
    ![](_v_images/20210405001054114_31570.png =627x)
  2. 我上传到文件/opt下
  3. 安装前要先安装编译环境，yum -y install gcc make gcc-c++ openssl-devel kernel-devel
  4. 解压文件 tar -xvf node-v14.16.0-linux-x64.tar.xz
  5. 运行命令node，进入node.js命令窗口则表示安装成功
  6. 如不成功，配置环境变量
    打开文件 vim /etc/profile
    在文件的底部增加 export PATH=$PATH:/opt/nodejs/bin
  7. source /etc/profile 运行命令使得环境变量立即生效
