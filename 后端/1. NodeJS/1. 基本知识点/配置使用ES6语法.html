配置使用ES6语法

初始化项目

  • mkdir server && npm init 生成package.json文件

安装使用的工具包三个

  • npm install babel-cli
  • npm install babel-core
  • npm install babel-preset-2015

配置文件

  • 在根目录下新建文件 .babelrc，并配置文件内容

配置package.json文件

  • 在文件的跟属性中添加type字段，默认该字段值为commonJS,且默认不显示，配置属性值为module

配置完成

  • 以上配置完成后即可使用ES6语法
  • 使用import * from ‘lodash’;
