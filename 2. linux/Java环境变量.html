Java环境变量

linux安装包 直接解压

在
/etc/profile
文件中添加以下内容，java版本号根据实际进行更改

JAVA_HOME=/opt/java/jdk1.8.0_181
PATH=$JAVA_HOME/bin:$PATH
CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
export JAVA_HOME
export PATH
export CLASSPATH

保存，退出
使用命令
source /etc/profile
使环境变量生效
